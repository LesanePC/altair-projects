# Altair — корпоративный сайт агентства недвижимости

Корпоративный сайт агентства недвижимости **Альтаир Недвижимость**, разработанный для действующего бизнеса. Сайт предоставляет клиентам информацию об услугах, объектах недвижимости, преимуществах компании и способах связи.

## 🚀 Основные возможности

### Структура
- Многостраничная структура: главная, объекты недвижимости, услуги, о компании, контакты
- Полностью адаптивная и кроссбраузерная вёрстка
- Удобная навигация и структурированное меню

### Функционал
- **Объекты недвижимости** — карточки с фильтрацией по категориям, галерея изображений, избранное
- **Услуги** — основные и юридические услуги с ценами и описанием
- **О компании** — история, миссия, команда специалистов, достижения
- **Контакты** — интерактивная карта, виджет отзывов, форма обратной связи
- **Формы обратной связи** — с валидацией и маской телефона
- **Слайдер отзывов** — автопрокрутка с точками навигации
- **Анимации** — плавное появление элементов при скролле
- **Кнопка "Наверх"** — для удобной навигации

### Технические особенности
- Семантическая HTML-разметка
- SEO-оптимизация (мета-теги, Open Graph, микроразметка Schema.org)
- Поддержка тёмной темы (prefers-color-scheme)
- Стили для печати
- Доступность (ARIA-атрибуты, skip-link, focus-стили)
- Быстрая загрузка и стабильная работа на всех устройствах

## 🧰 Используемые технологии

| Технология | Назначение |
|------------|------------|
| **HTML5** | Семантическая, SEO-дружественная разметка |
| **CSS3** | Flexbox, Grid, адаптивные медиа-запросы, анимации |
| **JavaScript (ES6+)** | Интерактивные элементы, фильтрация, слайдеры, формы |
| **Responsive Web Design** | Корректная работа на телефонах, планшетах и десктопах |
